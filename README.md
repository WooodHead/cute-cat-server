### 사용 기술

- 프론트엔드
    - Typescript, React, react-material-ui, axios
- 백엔드
    - Typescript (Node.js), NestJS, Typeorm
- 인프라
    - AWS ECS, Docker, AWS RDS, AWS S3, Mysql, Github (with Github Actions)
    - Route53, ACM

### 주요 기능

1. 회원가입/로그인 기능
2. 포스트 조회/생성/수정/삭제 기능
3. 포스트에 들어갈 사진 업로드 기능
4. 페이스북 페이지에 포스트 게시 기능
